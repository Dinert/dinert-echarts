我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 Dinert Amap

用法 ​

shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install echarts lodash @dinert/echarts

# Yarn
$ yarn add echarts lodash @dinert/echarts

# pnpm
$ pnpm install echarts lodash @dinert/echarts
